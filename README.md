## Building
You'll need [PHP](https://www.php.net/) and [Composer](https://getcomposer.org/) installed on your computer in order to build this app.

```bash
$ git clone https://github.com/eric-souzams/symply-events
$ cd symply-events
$ php artisan serve
```
